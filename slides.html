Firebase Auth

  • Identity-as-a-Service

  • Pre-built UI components

Firebase Storage

  • Store and serve media content
  • Scales automatically
  • Serve content based on rules

Cloud FireStore

  • Store and sync data in real-time
  • NoSQL, document collections structure
  • Offline handling and auto scaling

Firebase Hosting

  • Fast and secure web hosting
  • Backed by global CDN
  • Custom domain
